Foundations

What It Is & Where You Use It

Claude Code lives in your terminal. It understands your codebase. It works everywhere you code.

💻 Terminal

Full-featured CLI — the original interface.

🔌 IDE Extensions

VS Code, Cursor, JetBrains IDEs.

🌐 Browser

claude.ai/code — zero setup.

Not a chat window you copy-paste from. It reads your files, edits them, runs commands, manages git — all directly in your environment.

The Secret Sauce

Why It Works So Well

Two things set it apart

1. Understands Your Codebase

It doesn't just read the file you point at. Claude Code explores your project, follows imports, understands patterns, and knows how components relate. Ask it to fix a bug — it finds the relevant files itself.

2. Learns How You Work

Memory, rules, hooks, MCP integrations. Everything compounds in value the more you invest in it. It doesn't just get better — it gets more yours.

The Secret Sauce

It Learns Through Layered Config

CLAUDE.md

  • Project (shared with team)

Plus: MCP integrations, hooks, skills, plugins. Everything at user, project, or local scope.
